### 1. 🎯 **Prioridades Comunitarias**
- Votar en issues de GitHub
- Participar en encuestas de features
- Reportar bugs y sugerir mejoras

### 2. 👥 **Roles de Contribución**
| Rol | Responsabilidades | Cómo Empezar |
|-----|-------------------|--------------|
| **Traductor** | Traducción de contenido | Unirse al equipo i18n |
| **Tester** | Testing de nuevas features | Ejecutar testing suite |
| **Developer** | Desarrollo de código | Pick up "good first issue" |
| **Documentador** | Mejora de documentación | Actualizar docs existentes |

### 3. 📅 **Proceso de Desarrollo**
```
1. Propuesta de feature → Issue en GitHub
2. Discusión comunitaria → Comments + votes
3. Especificación → PR con detalles técnicos
4. Desarrollo → Asignado a agente/especialista
5. Testing → Suite automatizada + testing manual
6. Deploy → Staging → Production
7. Retrospectiva → Lecciones aprendidas
```
